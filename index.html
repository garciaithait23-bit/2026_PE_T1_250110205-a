Análisis de Datos con Python

Guía teórica y práctica para la representación gráfica de variables estadísticas.

1. Datos Cuantitativos Discretos

Representan cantidades contables con valores enteros. En estadística, se utilizan para analizar distribuciones de frecuencia de eventos puntuales.

Gráfica de Barras: Rendimiento Académico

Contexto: Medición del hábito lector en un grupo de estudiantes.

Uso Estadístico: Comparación de frecuencias absolutas entre categorías nominales u ordinales discretas.

import matplotlib.pyplot as plt
estudiantes = ["Ana", "Luis", "Carlos", "Marta", "Sofía"]
libros = [1, 3, 2, 4, 2]
plt.bar(estudiantes, libros)
plt.title("Libros leídos por estudiante")
plt.show()

Gráfica de Línea: Cumplimiento de Tareas

Contexto: Seguimiento diario de entrega de deberes durante una semana escolar.

Uso Estadístico: Análisis de tendencias temporales en datos discretos para observar patrones de comportamiento.

import matplotlib.pyplot as plt
dias = [1, 2, 3, 4, 5, 6]
tareas = [2, 4, 3, 5, 4, 6]
plt.plot(dias, tareas, marker='o')
plt.title("Tareas entregadas por día")
plt.show()

Gráfica de Pastel: Resultados Electorales

Contexto: Distribución de votos obtenidos por candidatos en una elección.

Uso Estadístico: Representación de la frecuencia relativa y la composición porcentual de un total (espacio muestral).

import matplotlib.pyplot as plt
candidatos = ["A", "B", "C", "D"]
votos = [10, 15, 8, 12]
plt.pie(votos, labels=candidatos, autopct="%1.1f%%")
plt.title("Distribución de votos")
plt.show()
2. Datos Cuantitativos Continuos

Resultan de mediciones y pueden incluir decimales. Su análisis se enfoca en la densidad y dispersión de los datos.

Histograma: Distribución de Estaturas

Contexto: Recopilación de medidas biométricas en una población juvenil.

Uso Estadístico: Agrupación en intervalos de clase para identificar la forma de la distribución (normalidad o sesgo).

import matplotlib.pyplot as plt
estaturas = [1.60, 1.72, 1.68, 1.75, 1.80, 1.65, 1.70, 1.78, 1.69, 1.74]
plt.hist(estaturas, bins=5, edgecolor='black')
plt.title("Distribución de estaturas")
plt.show()

Gráfica de Línea: Variación Térmica

Contexto: Registro de temperatura ambiental por hora.

Uso Estadístico: Monitoreo de variables continuas en una serie de tiempo para detectar fluctuaciones cíclicas.

import matplotlib.pyplot as plt
horas = [1, 2, 3, 4, 5, 6]
temperaturas = [22.5, 23.0, 24.3, 25.1, 26.4, 27.2]
plt.plot(horas, temperaturas, marker='o', color='orange')
plt.title("Temperatura por hora")
plt.show()

Gráfica de Dispersión (Scatter): Correlación

Contexto: Relación entre el peso y la estatura de un grupo de individuos.

Uso Estadístico: Identificación del grado de correlación entre dos variables continuas, base para modelos de regresión.

[Image of a scatter plot showing positive correlation] 
import matplotlib.pyplot as plt
estatura = [1.60, 1.65, 1.70, 1.75, 1.80]
peso = [55.2, 60.5, 68.0, 72.3, 80.1]
plt.scatter(estatura, peso)
plt.title("Relación entre estatura y peso")
plt.show()
3. Datos Cualitativos

Describen cualidades o categorías no numéricas. Se analizan mediante la frecuencia de aparición de cada atributo.

Gráfica de Barras: Estudio de Mercado

Contexto: Preferencia de colores en un grupo focal.

Uso Estadístico: Identificación de la Moda en variables nominales.

import matplotlib.pyplot as plt
categorias = ["Azul", "Rojo", "Verde"]
frecuencia = [4, 2, 2]
plt.bar(categorias, frecuencia)
plt.title("Color favorito")
plt.show()

Gráfica de Pastel: Modos de Transporte

Contexto: Tipo de transporte utilizado para ir al colegio.

Uso Estadístico: Visualización de la partición de categorías dentro de una población determinada.

import matplotlib.pyplot as plt
categorias = ["Bus", "Auto", "Bicicleta"]
frecuencia = [4, 2, 2]
plt.pie(frecuencia, labels=categorias, autopct="%1.1f%%")
plt.title("Tipo de transporte")
plt.show()

Gráfica de Barras Horizontal: Lealtad de Marca

Contexto: Marcas de celulares más utilizadas entre usuarios encuestados.

Uso Estadístico: Mejora de la legibilidad en comparaciones de categorías cuando hay nombres largos o múltiples etiquetas.

import matplotlib.pyplot as plt
categorias = ["Samsung", "Apple", "Xiaomi"]
frecuencia = [4, 2, 2]
plt.barh(categorias, frecuencia)
plt.title("Marca de celular más utilizada")
plt.show()
